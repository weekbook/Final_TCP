# TCP_Chat_Project
1:1채팅이 아니라 복수의 사용자들이 접속하여 메시지를 전송하는 즉시 모든 사용자들에게 
보여줄 수 있는 채팅 프로그램 입니다. 또한 스레드를 사용하여 메인 스레드는 서버의 작동 및 연결을
담당하고 다른 스레드는 송/수신을 담당하게 만들어 프로젝트를 진행하였습니다.
